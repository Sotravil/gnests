# Changelog

All notable changes to GNESTS are documented in this file.

The format follows semantic versioning: `MAJOR.MINOR.PATCH`.

## [0.8.0] - 2026-03-12

### Added
- GitHub-referenced version source using `version.json`.
- In-app changelog section in `index.html` with release date and release highlights.
- Auto update notification for newer versions with click routing to changelog section (`?changelog=1`).
- Header release badges: local app version, latest published version, and launch date.

### Changed
- Updated README structure with project categories, release policy, and GitHub release workflow.
- Improved release transparency by linking runtime update checks to repository metadata.

## [0.7.0] - 2026-03-12

### Added
- Professional command modules and action execution pipeline.
- Preset, draft, asset, embed local store modules.
- Embed runtime output and docs (`api-contract.md`, `command-reference.md`, schemas).

### Changed
- Expanded `index.html` professional console into a multi-panel operator workspace.

## [0.1.0] - 2026-03-12

### Added
- Initial static notification lab scaffold.
- Service worker registration and local scheduling flow.
